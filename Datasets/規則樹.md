## BOM 規則

### 1. 階層規則
| Level | 說明 | 用量單位 |
|-------|------|----------|
| 1 | 成品包裝材料 | PCS/箱 |
| 2 | 半成品/子組件 | PCS/組 |
| 3 | 基礎零件 | PCS |

### 2. 編碼規則
- **Part No. 規則**: [分類碼][規格碼][序號]
  - P開頭: 包裝材料 (P+材質+尺寸+序號)
  - M開頭: 五金/機構件
  - 數字開頭: 電子零件
  - C開頭: 連接器
  - W開頭: 線材
  - S開頭: 螺絲/螺帽

### 3. Part Type 分類
| 類別 | 包含類型 |
|------|----------|
| 包裝材 | 外箱, 隔板, 珍珠棉, 牛皮盒, 棧板, 紙護角 |
| 電子零件 | IC, 電阻, 電容, 二極體, 電晶體, 光耦, LED, PCB |
| 機構件 | 散熱片, 風扇, 風扇架, 螺絲, 螺柱, 絕緣片 |
| 線材 | AC線, DC線, 端子線, 排線 |
| 其他 | 保險絲, NTC, 標籤, 壓敏電阻 |

### 4. 用量規則 (QPA)
- QPA = 該零件在上一層組件中的用量
- Level 1 QPA: 以整機為單位
- Level 3 QPA: 以該 Level 2 子板為單位

### 5. 衍生機種規則
- 同一 PCB 可衍生多種型號 (C22B01, S25404 等)
- 變更零件時需更新 BOM 版本
- 壓敏電阻空 (14D471K 預留位置)

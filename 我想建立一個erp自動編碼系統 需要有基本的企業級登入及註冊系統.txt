我想建立一個erp自動編碼系統 需要有基本的企業級登入及註冊系統
技術棧:
前端: vite@latest + React + TypeScript + React Compiler
後端: uv + python + flask
資料庫: PostgreSQL主資料庫 + Redis快取資料庫
LLM服務: 暫定
容器化及CICD: git-workflow, git, docker, docker-compose

登入/註冊系統(admin帳號由資訊人員管理)
Admin帳號由資料庫播種 (Database Seeding) + 環境變數 (ENV)建立
要做Admin-Approved/Registration然後Closed System/Invite-Only Alternative
一般使用者註冊->資料庫建立帳號被標記被審查->Admin收到通知並審核通過使用者才正式獲得對應的 RBAC（Role-Based Access Control）角色權限
admin要設定Rate Limiting & Lockout, 隱藏後台入口與 IP 限制, Audit Logs
admin部分
專案初始化由資料庫播種 (Database Seeding) + 環境變數 (ENV)建立
直接在登入頁面登入即可進入admin版本的前端(功能=一般使用者的功能+admin功能(因此前端可共用一般使用者))

登入部分
需要帳號oremail&密碼 才能登入
可以選擇忘記密碼 忘記密碼流程如下
一般使用者於登入頁面點選忘記密碼 -> 輸入帳號或email -> 於現實世界報備資訊人員(admin) -> admin登入系統在管理人員頁面點選一般使用者忘記密碼確認按鈕輸入temp密碼送出更改
-> 一般使用者返回登入頁面用帳號oremail&temp密碼登入後出現更改密碼頁面 -> 一般使用者依據頁面提示輸入自己的新密碼和確認新密碼後進入系統 -> 下次登入使用新密碼

註冊部分
一般使用者註冊->資料庫建立帳號被標記被審查->Admin收到通知並於管理人員頁面審核通過使用者才正式獲得對應的 RBAC（Role-Based Access Control）角色權限
